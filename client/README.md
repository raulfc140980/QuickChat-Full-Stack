How to Setup and Run my Project

1. Clone GitHub project from this web address: https://github.com/raulfc140980/QuickChat-Full-Stack or from the same web address download as zip the project, unzip it and then open the folder with VS Code, Antigravity or other code editor

2. Setup MongoDB and obtain Mongo URI
Link : https://www.mongodb.com/cloud/atlas/register
Add mongoDB URI in “ .env ” file of server folder

3. Setup Cloudinary
Link : https://cloudinary.com/users/register_free
Add Cloudinary Keys ( cloud name, secret key, api key ) in “ .env ” file of server folder

Cloudinary is a cloud-based service that helps developers and businesses store, manage, optimize, and deliver images and videos for websites and apps.

4. Open server folder in integrated terminal

5. Install dependencies using command

npm install

6. Run project using command

npm run dev

7. To see the project deployed online and interact with it open the link:

https://quick-chat-client-eight-alpha.vercel.app/login
